# Titanic Survival Analysis

## Description
This project analyzes the Titanic dataset to find factors affecting passenger survival using data analysis and machine learning.

## Dataset
- **Source:** Kaggle  
- **Description:** Includes passenger details like age, gender, class, and survival status.

## Steps Performed
1. Data Cleaning  
2. Exploratory Data Analysis  
3. Visualization  
4. Model Building  

## Results
- Higher survival rate for females  
- First-class passengers had better survival chances  
- Age and class impacted survival  

## Tools Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

## Conclusion
The project highlights key factors influencing survival and demonstrates basic predictive modeling.
