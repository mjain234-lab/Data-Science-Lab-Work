# Snake Game using Python

## Description
This is a GUI-based Snake Game developed using Python and Tkinter. The player controls the snake to eat food, grow in size, and avoid collisions.

## Dataset
- Not applicable

## Steps Performed
1. Game design using grid system  
2. Snake movement implementation  
3. Food and bonus generation  
4. Collision detection  
5. Score and high score tracking  

## Results
- Smooth working game  
- Difficulty levels implemented  
- High score saved using JSON  

## Tools Used
- Python  
- Tkinter  
- JSON  
- Random  

## Conclusion
This project demonstrates game logic, GUI development, and event handling in Python.
