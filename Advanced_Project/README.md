# Cybersecurity Risk Analysis

## Description
This project focuses on analyzing cybersecurity incidents to identify patterns, trends, and risk factors. It helps in understanding different types of cyberattacks and predicting potential risks.

## Dataset
- **Source:** Kaggle / Public dataset  
- **Description:** Contains data on cyberattacks including attack type, year, target, and severity.

## Steps Performed
1. Data Cleaning  
2. Exploratory Data Analysis  
3. Visualization  
4. Model Building (Logistic Regression, Random Forest, SVM, ANN)

## Results
- Identified major attack types  
- Observed trends over time  
- Achieved high model accuracy  

## Tools Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

## Conclusion
The project provides insights into cybersecurity threats and helps in better risk management.
